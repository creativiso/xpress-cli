# Installation

## Prerequisites

- Node.js ≥ 20.0.0

## Install globally

```bash
npm install -g @creativiso/xpress-cli
```

## Verify

```bash
xpress --version
xpress --help
```

## Update

```bash
npm update -g @creativiso/xpress-cli
```

## Uninstall

```bash
npm uninstall -g @creativiso/xpress-cli
```
