# Changelog

## 0.1.0 — 2026-05-24

Initial release.

- Full CRUD for pages, articles, article-categories, products, product-categories, product-attributes, product-variations, product-options, orders
- Read-only access to strings and indexed-urls
- `products raw-attrs` sub-group for raw attribute value management
- `product-attributes values` sub-group for predefined value management
- Global `--json`, `--quiet`, `--verbose`, `--token`, `--url`, `--lang` flags
- Config stored at `~/.xpress-cli.json` with `0600` permissions
